1. **Go to Settings >** [Integrations](https://app.incident.io/settings/integrations)

2. **Press "Install" next to Opslevel**

3. **Fetch your API key from Opslevel**

You'll need to create an API key by heading to the settings page of [your Opslevel account](https://app.opslevel.com/api_tokens) .

4. **Press "Save"**

5. **Go to** [Catalog](https://app.incident.io/catalog) **to view connected Catalog entities**
