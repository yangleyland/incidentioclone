1. **Go to Settings >** [Integrations](https://app.incident.io/settings/integrations)

2. **Press "Install" next to Cortex**

3. **Fetch your API key from Cortex**

You'll need to create an API key by heading to the settings page of [your Cortex account](https://app.getcortexapp.com/login) .

4. **Press "Save"**

5. **Go to** [Catalog](https://app.incident.io/catalog) **to view connected Catalog entities**
