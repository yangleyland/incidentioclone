Of course!

Should you wish to uninstall the Zoom app, you can visit > [https://app.incident.io/api/zoom_uninstall](https://app.incident.io/api/zoom_uninstall) , which will revoke the existing credentials and clear any Zoom data from [incident.io](http://incident.io/) systems.
