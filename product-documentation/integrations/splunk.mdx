# Splunk

There are currently 2 ways of integration with Splunk from incident.io:

- For using **Splunk On-Call** and escalating incidents or paging policies [please view our documentation here.](https://help.incident.io/en/articles/6679445-splunk-on-call-victorops)

- For using **Splunk SIEM** to stream your audit logs [please view our audit log documentation here.](https://help.incident.io/en/articles/7339034-audit-logs)
